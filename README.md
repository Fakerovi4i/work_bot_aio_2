# 📌 Описание
### Бот-помощник для рабочих процессов:

- Получение и удаление задач от администратора
- Отметка прихода/ухода
- Отправка отчётов (с обязательными полями)
- Отправка предложений и идей
- Общение с ИИ (Mistral AI)


- 👨‍💻 Админ-панель: рассылки, бан/разбан, просмотр отчётов и идей, добавление задач


## 🛠 Установка

1. Клонируйте репозиторий
2. Установите зависимости

`pip install -r requirements.txt`

- Создайте файл .env в корне:


- TOKEN=ваш_токен_бота

- API_KEY=ключ_mistral_api

- API_MODEL=mistral-small-latest

### 🚀 Запуск: `python main.py`



## 📁 Структура

- main.py – точка входа
- loader.py – инициализация бота, сессии, планировщика
- database.py – работа с SQLite (таблицы: users, tasks, doc, improve, admin и др.)
- keyboards.py – инлайн-клавиатуры
- states_group.py – FSM-состояния
- handlers/ – обработчики (user, admin, ai))

utils.py – планировщик (напоминания)


## ⚙️ Основные команды
- /start – главное меню
- /admin – вход в админ-панель (только для добавленных админов)
- /addadmin – назначить администратора
- /delladmin – снять права администратораа

###
### 📝 Примечания

- Отчёты принимаются только с тремя обязательными пунктами (проверка на стороне бота).
- ИИ использует контекст до 20 сообщений, история хранится в FSM.
- Логирование: INFO в консоль, ERROR в файл bot.log.
- Сессия aiohttp одна для всех запросов (избегает ошибок прокси)..
