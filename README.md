# 🤖 Corporate Assistant Bot

**Телеграм-бот для автоматизации рабочих процессов в компании.**

[![GitHub](https://img.shields.io/badge/GitHub-Repository-blue?style=for-the-badge&logo=github)](https://github.com/Fakerovi4i/work_bot_aio_2)

Помогает сотрудникам получать задачи, отмечать приход/уход, отправлять отчёты и предложения. Администраторы получают полноценную панель управления.

### ✨ Основные возможности
- Полноценная **админ-панель** (добавление задач с фото, рассылки, бан/разбан, управление админами)
- Система задач + удаление задач пользователями
- Отметка прихода/ухода + автоматические напоминания
- Отправка отчётов по строгому шаблону (валидация на стороне бота)
- Внесение предложений и идей
- Встроенный **ИИ-помощник** (Mistral AI) с сохранением контекста
- Логирование действий и обработка сетевых ошибок

### 🛠 Технологии
- **Python 3.11** + **aiogram 2.x**
- Finite State Machine (FSM)
- SQLite (структурированная работа через класс Database)
- APScheduler (планировщик задач)
- aiohttp + Mistral AI API
- Логирование (console + файл)

### ⚡ Сложности и как я их решил
- Огромный монолит → разбил на модули
- Одновременная работа нескольких состояний и клавиатур → грамотно организовал `StatesGroup`
- Работа планировщика и напоминаний → интегрировал APScheduler с обработкой исключений
- Эффективная работа с базой данных → разделил логику по модулям

```bash

pip install -r requirements.txt

cp .env.example .env

```
